## العربية

نحن في 1SalemBOT نحترم خصوصيتك ونلتزم بحماية بياناتك.

عند استخدام البرنامج قد يتم جمع بيانات تشغيل محدودة بهدف تحسين استقرار البرنامج وتقديم الدعم الفني.

البيانات التي قد يتم جمعها:

* معرف التثبيت (Install ID)
* اسم قناة البث
* اسم حساب البوت
* إصدار البرنامج
* إصدار نظام التشغيل
* معلومات تشخيصية متعلقة بالأخطاء والأداء

لا يقوم البرنامج بجمع أو إرسال:

* كلمات المرور
* رموز OAuth الخاصة بالحسابات
* مفاتيح API الخاصة بالمستخدم
* بيانات الدفع
* الرسائل الخاصة
* محتوى المحادثات الشخصية

قد يستخدم البرنامج خدمات طرف ثالث مثل:

* Twitch
* Kick
* YouTube
* Supabase
* OpenAI
* VLC

ويخضع استخدام تلك الخدمات لسياسات الخصوصية الخاصة بها.

يتم استخدام البيانات المجمعة للأغراض التالية:

* تحسين أداء البرنامج
* إصلاح الأخطاء
* قياس عدد التثبيتات النشطة
* تشخيص المشاكل التقنية

يتم توفير البرنامج "كما هو" دون أي ضمانات صريحة أو ضمنية.

باستخدامك للبرنامج فإنك توافق على هذه السياسة.
